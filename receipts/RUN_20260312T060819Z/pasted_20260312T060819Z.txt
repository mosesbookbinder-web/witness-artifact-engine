Let

* (X) be a set of states
* (\Sigma) be a set of signals
* (A) be a set of artifacts
* (W) be a set of witnesses
* (T) be a set of transforms
* (D) be a set of decoders
* (M) be a set of metadata or bindings

Define an artifact as a tuple
[
a=(\sigma,m)\in A:=\Sigma\times M.
]

Define a witness map
[
\omega:A\to W.
]

Define a signal extraction map
[
s:A\to \Sigma,\qquad s(\sigma,m)=\sigma.
]

Define a binding map
[
b:A\to B
]
where (B) is a binding space, for example hashes, provenance records, replay bundles, or other invariants.

Define a decoder
[
\delta:D\times A\to Y
]
into some interpretation space (Y).

The core requirement is that admissibility is not based on surface form, but on preservation of signal under allowed transforms.

Let
[
\tau:T\times A\to A
]
be a transform.

Then signal preservation under transform (\tau) means
[
s(\tau(t,a)) \sim s(a)
]
for an equivalence relation (\sim) on (\Sigma).

The minimal atomic object is not the raw signal alone but the preserved, witnessed signal unit:
[
\alpha(a):=(s(a),,b(a),,\omega(a)).
]

Call (\alpha(a)) the witness atom.

A transform (t\in T) is admissible for artifact (a) iff the witness atom is preserved up to declared equivalence:
[
\alpha(\tau(t,a)) \equiv \alpha(a).
]

More explicitly, define
[
\alpha(\tau(t,a)) \equiv \alpha(a)
]
iff all of the following hold:

[
s(\tau(t,a)) \sim s(a),
]
[
b(\tau(t,a)) \cong b(a),
]
[
\omega(\tau(t,a)) \approx \omega(a),
]
where (\sim,\cong,\approx) are declared equivalence relations for signal, binding, and witness respectively.

Now define continuation.

Let
[
C:A\to { \mathrm{HALT},\mathrm{INDETERMINATE},\mathrm{PASS}}.
]

Given a set of required predicates
[
\mathcal{P}={P_1,\dots,P_n},
]
define

[
C(a)=
\begin{cases}
\mathrm{HALT}, & \exists i; H_i(a)=1,[4pt]
\mathrm{INDETERMINATE}, & \neg\exists i; H_i(a)=1 \text{ and } \exists j; U_j(a)=1,[4pt]
\mathrm{PASS}, & \text{otherwise,}
\end{cases}
]
where (H_i) are hard failure predicates and (U_j) are unknown or missing predicates.

A basic hard-failure set is

[
H(a)=
\bigl[
\neg \mathrm{parse}(a)
;\vee;
\neg \mathrm{stable}(b(a))
;\vee;
\neg \mathrm{replay}(a)
;\vee;
\neg \mathrm{authentic}(\omega(a))
\bigr].
]

A basic indeterminate set is

[
U(a)=
\bigl[
\mathrm{missing_provenance}(a)
;\vee;
\mathrm{missing_receipt}(a)
;\vee;
\mathrm{missing_bundle}(a)
\bigr].
]

Then the governing principle is:

[
\boxed{
\text{continuation is admissible iff the witness atom is preserved under declared transform}
}
]

or equivalently,

[
\boxed{
C(a)=\mathrm{PASS}
\iff
\bigwedge_{k} P_k(a)=1
}
]

with the predicates chosen so that they test signal preservation, not costume.

To make this domain-agnostic, define a projection family
[
\pi_j:A\to A_j,\qquad j\in J
]
for domains, languages, formats, or coordinate systems.

Then require projection consistency:

[
\forall j\in J,\quad s_j(\pi_j(a)) \sim_j s(a),
]
and
[
\forall j\in J,\quad \alpha_j(\pi_j(a)) \equiv_j \alpha(a).
]

So a single object may appear differently across domains, but continuation is governed by whether the underlying witness atom remains invariant under projection.

That is the mathematical core.

A compact form is:

[
a=(\sigma,m),\qquad
\alpha(a)=(s(a),b(a),\omega(a)),
]
[
t \text{ admissible } \iff \alpha(\tau(t,a))\equiv \alpha(a),
]
[
C(a)=
\begin{cases}
\mathrm{HALT}, & \text{hard violation},\
\mathrm{INDETERMINATE}, & \text{missing witness},\
\mathrm{PASS}, & \text{signal preserved}.
\end{cases}
]

And the central axiom is:

[
\boxed{
\text{Preservation of signal is the necessary condition for admissible continuation.}
}
]
